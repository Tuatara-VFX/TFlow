# Supported pipelines

TFlow can be used in combinaison with the following pipelines.

| Pipeline            | State |
|-------              |-------|
| Unity URP           |  ✅  |
| Unity HDRP          |  ✅  |
| Unity Amplify       |  ✅  |
| Unity VFX Graph     |  ✅  (supported by default) |
| Unity Shader Graph  |  ✅   |
| Unity Custom Shaders  |  ✅   |
| Unreal Engine       |  Work in progress  |
